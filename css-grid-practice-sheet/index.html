CSS Grid

The CSS grid is a newer standard that makes it easy to built complex resposive layouts. It works by turning an HTML element into a grid, and lets you place child elements anywhere within.

In this course, you'll learn the fundamentals of CSS grid by building different complex layouts, including a blog.
